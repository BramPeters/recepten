Een nieuwe site maken in Dreamweaver

In DreamWeaver kan je een website als geheel opzetten. Het voordeel is dat alle benodigde pagina’s en figuren bij elkaar gehouden worden. Een ander voordeel is het gemak waarmee je de website kunt op de webserver zetten en later bijwerken.

Site – Manage Sites - New:
de Wizard Site definition verschijnt, je beantwoordt een aantal vragen
“Hoe wil u de website noemen?”
vul in George’s site
“Wil u Server technologie gebruiken?”
Neen, ga verder
“Hoe wil u tewerk gaan met de bestanden?”
kies voor Ik wil de bestanden lokaal bewerken en dan uploaden naar de server (aangeraden)
”Waar wil u de bestanden lokaal opslaan?
Op C:\My Documents\George\George’s site (wordt automatisch ingevuld)
“Hoe wil u de bestanden kopiëren naar de Remote Server?”
kies voor Local/Network
”In welke map op die server wil u de bestanden kopiëren?”

Neen, ga verder
