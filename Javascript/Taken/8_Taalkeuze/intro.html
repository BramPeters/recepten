Javascript PF taak: Taalkeuze

Opdracht:

  • Maak een intropagina waar de gebruiker d.m.v.hyperlinks een definitieve taalkeuze kan maken.
  • Door op deze link te klikken wordt hij naar een taalpagina gebracht in de respectievelijke taal.
  • Bij een tweede bezoek krijgt de gebruiker de intropagina niet meer te zien.
  • Op elk van de pagina's is er echter een link voorzien die hem kan terugbrengen naar de intropagina.
  • Voorzie ook een hyperlink/knop vernietig cookie voor testing
Welkom beste bezoeker. Kies uw taal door op een vlaggetje te klikken.
